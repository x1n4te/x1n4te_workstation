# First Look
There are two open ports which are SSH and HTTP. SSH is not a part of this problem, the HTTP is running Node.JS with a templater engine "handlebar". Handlebar has numerous vulnerabilities, but in this case we are going to use SSTI to use Remote Code Execution and create a shell inside.
