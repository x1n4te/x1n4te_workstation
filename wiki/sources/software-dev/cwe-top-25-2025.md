## WIMS-BFP Mitigation

| CWE | WIMS-BFP Relevant Code | Mitigation |
|---|---|---|
| CWE-79 (XSS) | Next.js frontend, user inputs | React auto-escapes, but verify all dangerouslySetInnerHTML |
| CWE-89 (SQLi) | psycopg2 queries | ✅ Parameterized queries by default |
| CWE-352 (CSRF) | FastAPI endpoints | ✅ JWT auth (stateless), no cookies |
| CWE-862 (Missing AuthZ) | All API endpoints | ✅ RLS + JWT validation + FRS role checks |
| CWE-78 (OS Cmd) | Celery tasks, subprocess calls | ⚠️ Audit any subprocess/shell usage |
| CWE-502 (Deserialization) | JSON parsing | ✅ Pydantic validation, no pickle |
| CWE-918 (SSRF) | Web scraping, external APIs | ⚠️ Validate URLs, restrict outbound |
| CWE-306 (Missing Auth) | DMZ endpoints | ✅ DMZ has separate auth flow |
| CWE-770 (No Limits) | API rate limiting | ⚠️ Verify rate limiting on all endpoints |
