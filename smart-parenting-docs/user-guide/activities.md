# Activities

The Activities tab is where you record what your child has done and plan what they'll do next. It has two modes: **Log** (for past activities) and **Schedule** (for future reminders).

---

## Switching Modes

At the top of the screen, a pill toggle lets you switch between:
- **Log** — Record something that already happened
- **Schedule** — Plan a future activity with reminders

---

## Log Mode

### 1. Choose an Activity Type

Tap one of six cards:
- 📱 **Screen Time** — TV, tablet, phone, computer
- 🌙 **Sleep** — Nighttime sleep
- 😴 **Nap** — Daytime sleep
- 🍳 **Meals** — Breakfast, lunch, snack, dinner
- 🏊 **Physical Activity** — Running, swimming, outdoor play
- 📚 **Learning** — Reading, homework, educational games

### 2. Enter Time

**For time-range activities** (sleep, nap, screen, physical, learning):
- Use the **Start Time** and **End Time** inputs
- Each has hour/minute steppers and an AM/PM toggle
- The app automatically calculates the duration and displays it (e.g., "= 8h 30m")
- If the end time is earlier than the start time, the app assumes it crossed midnight

**For meals:**
- Use the **Single Time** input to record when the meal happened

### 3. Add Details

Depending on the activity type, you'll see additional options:

| Activity | Details You Can Add |
|----------|---------------------|
| Screen Time | Device (TV, tablet, phone) and category (leisure or educational) |
| Sleep / Nap | Quality rating (poor, fair, good, excellent) |
| Meals | Meal type, food groups (fruits, vegetables, protein, grains, dairy), quality rating |
| Physical Activity | Activity name (swimming, running, etc.) |
| Learning | Subject (reading, math, science, etc.) |

### 4. Add Notes (Optional)

A multiline text box lets you jot down anything extra — "Fell asleep quickly after story time" or "Ate all vegetables."

### 5. Submit

Tap **Log Activity**. A green success screen appears briefly, then the app returns to the previous tab.

---

## Schedule Mode

Use this to plan future activities and get reminder notifications.

### 1. Choose an Activity Type

Same six types as Log mode.

### 2. Pick a Date

Use the date navigator (← → arrows) to select the day. Defaults to today.

### 3. Set Start Time

Use the hour/minute/AM-PM steppers to set when the activity begins.

### 4. Set Duration Range

For most activities, you define a **minimum** and **maximum** duration:
- Example: Nap for "1h — 2h"
- This creates a flexible window rather than a fixed duration

Meals don't have a duration range — just a single reminder time.

### 5. Add Category Details

- **Screen Time:** Choose "leisure" or "educational"
- **Meals:** Choose breakfast, lunch, snack, or dinner

### 6. Submit

Tap **Schedule Activity**. The app saves the schedule and instantly redirects you to the **History tab** so you can see it in your upcoming list.

> **Note:** If scheduling fails, you stay on the Activities tab with an error message so you can retry.

---

## Tips

- You must have a child selected to log or schedule. If not, the app prompts you to add one.
- The form resets automatically after a successful submit.
- You can tap the **create-outline** icon in the header to quickly start a new log.
