# History

The History tab is your read-only activity archive. Browse any day's logs, filter by activity type, and view charts showing trends over time.

---

## Calendar

A monthly calendar grid sits at the top:
- **Gray dots** beneath dates indicate days with logged activities
- **Tap any date** to see that day's activities below
- **Arrow buttons** (← →) navigate between months
- **"Today"** button jumps back to the current month and day
- Weeks start on **Monday** (Sunday appears in red text)

---

## Activity List

Below the calendar, a scrollable list shows all activities for the selected day:

Each card displays:
- A colored icon matching the activity type
- A human-readable description:
  - *"Screen time (leisure) — 1h 30m on phone"*
  - *"Sleep — 9h 0m (good)"*
  - *"Breakfast @ 7:30 AM — good · fruits, protein"*
- The time of day

Activities are sorted newest first.

---

## Filter Bar

A horizontal row of chips lets you filter the list and stats:
- **All** — everything (default)
- **Screen** — screen time only
- **Sleep** — sleep only
- **Nap** — naps only
- **Meals** — meals only
- **Active** — physical activity only
- **Learn** — education only

When you select a filter, both the activity list and the stats charts update.

---

## Stats Section

At the bottom of the screen, tap the **Stats** header to expand charts.

### Weekly View

Shows data for the Monday–Sunday week containing your selected date:
- **Bar charts** — one per activity type, showing hours per day
- **Pie chart** — food group distribution from meals that week

### Monthly View

Shows data for the entire viewed month:
- **Bar charts** — grouped by ~7-day chunks (e.g., "1–7", "8–14")
- **Pie chart** — food groups for the whole month

### Chart Behavior
- If an activity type has no data, its chart is hidden automatically
- The Y-axis always starts at 0 — no misleading truncated bars
- Meals show only the pie chart (no duration bars)

---

## Empty States

- **No child selected:** Prompt to add a child profile
- **No activities for the selected day:** "No activities this day — tap Log to record one"
- **No data for stats:** "Not enough data for statistics"

---

## Pull to Refresh

Swipe down to reload all activities from the server. This ensures you see the latest data if you logged something on another device.

---

## Important Notes

- History is **read-only** — you cannot edit or delete past logs from this screen.
- All activities for the selected child are fetched at once and filtered on your device, so scrolling is fast.
