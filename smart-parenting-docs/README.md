## What is Smart Parenting?

Smart Parenting is a mobile application that helps parents track their children's daily activities — screen time, sleep, meals, physical activity, and learning — and receive AI-powered insights and recommendations based on that data.

## Design Philosophy

The app is built around a few core principles:

1. **Respect the parent's time** — Logging an activity takes seconds, not minutes.
2. **Privacy first** — All data is protected by Row-Level Security; parents own their children's data.
3. **Opt-in, not opt-out** — Notifications and features are disabled by default; parents choose what they want.
4. **Actionable insights** — AI recommendations cite real data, not generic advice.
