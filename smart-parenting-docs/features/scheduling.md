# Scheduling

Scheduling lets you plan your child's future activities and receive reminder notifications when they're about to start or end.

---

## Creating a Schedule

1. Go to the **Activities** tab.
2. Switch to **Schedule** mode using the pill toggle.
3. Select an activity type (Screen Time, Sleep, Nap, Meals, Physical Activity, or Learning).
4. Pick a date using the navigator.
5. Set the **start time**.
6. Set a **duration range** (minimum to maximum). For example:
   - Nap: 1 hour — 2 hours
   - Screen Time: 30 minutes — 1 hour
7. Add category details if applicable:
   - Screen Time: leisure or educational
   - Meals: breakfast, lunch, snack, or dinner
8. Tap **Schedule Activity**.

After saving, the app automatically redirects you to the **History tab** so you can immediately see and confirm the new schedule.

---

## Flexible Duration

Unlike logging (which records exact times), scheduling uses a **range**:
- **Minimum duration:** The shortest you expect the activity to take
- **Maximum duration:** The longest you expect it to take

This is useful for activities with natural variation, like naps or outdoor play. The app uses this range to:
- Send a start reminder at the beginning
- Send an end-of-window reminder when the maximum duration is reached

Meals are the exception — they use a single reminder time rather than a range.

---

## Where Schedules Appear

Scheduled activities appear in three places:

1. **Dashboard** — In the "Upcoming Activities" section, sorted by start time
2. **History** — On the calendar day they occur, with a status badge
3. **Notification Shade** — As push reminders at the start and end times

---

## Managing Schedules

From the Dashboard's "Upcoming Activities" list, you can:

- **Log** — Mark the activity as completed (converts it into a real log entry)
- **Update** — Change the time or duration
- **Cancel** — Delete the schedule permanently

When you log a scheduled activity:
1. An activity record is created with the actual details
2. The schedule's status changes to **completed**
3. The associated reminders are cancelled

---

## Schedule Statuses

| Status | Meaning |
|--------|---------|
| **Pending** | Future activity, reminders active |
| **Completed** | Logged as an actual activity |
| **Skipped** | Marked as not happening |

---

## Tips

- You can schedule activities far in advance — there's no date limit.
- Overlapping schedules are allowed (e.g., screen time scheduled during nap time is permitted).
- If you delete a child, their pending schedules are also removed.
- Changing a routine time in Settings does **not** automatically update existing schedules.
